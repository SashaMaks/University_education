## Задание 1
### Дробная часть.
Точка координат.
Создайте сущность Точка, расположенную на двумерной плоскости, которая описывается:

- Координата Х: число
- Координата Y: число
- Может возвращать текстовое представление вида “{X;Y}”

Необходимо создать три точки с разными координатами и вывести на экран их текстовое
представление.

### Решение:
Создан класс Point в котором имеются:
#### поля
  1. `private double x;` - координата x
  2. `private double y;` - координата y
#### конструкторы 
  1. `public Point()` - для создания пустой точки в координатах 0;0
  2. `public Point(double x, double y)` - точка с координатами
  3. `public Point(Point point)` - точка из точки
#### геттеры
  1. `public double getX()` - возвращает координату x
  2. `public double getY()` - возвращает координату y
#### сеттеры для координат и для изменения точки полностью
  1. `public void setX(double x)` - устанавливает координату x
  2. `public void setY(double y)` - устанавливает координату y
  3. `public void setPoint(double x,double y)` - устанавливает координаты x и y 
#### преобразование в строку
1. `public String toString()` - выводит объект в виде “{X;Y}”
