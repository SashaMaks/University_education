## Задание 1
### Дробная часть.
Точка координат.
Создайте сущность Точка, расположенную на двумерной плоскости, которая описывается:

- Координата Х: число
- Координата Y: число
- Может возвращать текстовое представление вида “{X;Y}”

Необходимо создать три точки с разными координатами и вывести на экран их текстовое
представление.

### Решение:
Создан класс Point в котором имеются:
#### поля
  1. `private double x;` - координата x
  2. `private double y;` - координата y
#### конструкторы 
  1. `public Point()` - для создания точки в координатах 0;0
  2. `public Point(double x, double y)` - для создания точки по координатам
  3. `public Point(Point point)` - для создания точки по данной точке
#### геттеры
  1. `public double getX()` - возвращает координату x
  2. `public double getY()` - возвращает координату y
#### сеттеры
  1. `public void setX(double x)` - устанавливает координату x
  2. `public void setY(double y)` - устанавливает координату y
  3. `public void setPoint(double x,double y)` - устанавливает координаты x и y 
#### преобразование в строку
  1. `public String toString()` - выводит объект в виде “{X;Y}”

## Задание 2

#### Прямая.
Создайте сущность Линия, расположенную на двумерной плоскости, которая описывается:

- Координата начала: Точка
- Координата конца: Точка
- Может возвращать текстовое представление вида “Линия от {X1;Y1} до {X2;Y2}”

Для указания координат нужно использовать сущность Точка, разработанную в задании 1.1

#### Решение:
Создан класс Line в котором имеются:
- конструкторы принимаемые ничего, начальную и конечную точки, другую линию
- геттеры начала и конца линии
- сеттеры для установки начальной и конечной точки в виде как вставки точки так и координат
- преобразование в строку, которое выводит объект в виде “Линия от {X1;Y1} до {X2;Y2}”

Создан класс Point в котором имеются:
#### поля
  1. `private Point start;` - точка начала линии
  2. `private Point end;` - точка конца линии
#### конструкторы 
  1. `public Line()` - для создания линии начало и конец которой находятся в точке 0;0
  2. `public Line(Point start, Point end)` - для создания линии по заданным точкам
  3. `public Line(Line line)` - для создания линии по данной линии
#### геттеры
  1. `public Point getStart()` - возвращает точку начала линии
  2. `public double getY()` - возвращает точку конца линии
#### сеттеры для координат и для изменения точки полностью
  1. `public void setStart(Point start)` - устанавливает начало линии в данную точку
  2. `public void setStart(double x, double y)` - устанавливает начало линии в точку по координатам
  3. `public void setEnd(Point end)` - устанавливает конец линии в данную точку
  4. `public void setEnd(double x, double y)` - устанавливает конец линии в точку по координатам
#### преобразование в строку
  1. `public String toString()` - выводит объект в виде “Линия от { _ ; _ } до { _ ; _ }”

## Задание 3

#### Студент.
Создайте сущность Студент, которая описывается:

- Имя: строка
- Оценки: массив целых чисел.
- Может возвращать текстовое представление вида “Имя: [оценка1, оценка2,…,оценкаN]”

#### Решение: 
Создан класс Student в котором имеются:
#### поля
  1. `private String name;` - имя студента
  2. `private int[] grades;` - массив оценок студента
#### конструкторы 
  1. `public Student(String name)` - для создания студента без оценок, но с именем
  2. `public Student(String name, int... grades)` - для создания студента с именем и с хоть каким количеством оценок (или без них)
#### геттеры
  1. `public String getName()` - возвращает имя студена
  2. `public int[] getGrades()` - возвращает массив оценок студента
  3. `public int[] getGradesNoLinks()` - возвращает массив оценок без ссылки на студента у которого смотрят оценки
#### сеттеры
  1. `public void setName(String name)` - устанавливает новое имя студенту
  2. `public void setGrades(int[] grades)` - устанавливает новые оценки студенту
  3. `public void setGrade(int index, int grade)` - устанавливает по индексу новую оценку
#### преобразование в строку
  1. `public String toString()` - выводит объект в виде “имя: [оценка1, оценка2,…,оценкаN]”
