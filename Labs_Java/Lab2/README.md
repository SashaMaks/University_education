# Лабораторная работа №2
## Задание 1
#### Дробная часть.
Точка координат.
Создайте сущность Точка, расположенную на двумерной плоскости, которая описывается:

- Координата Х: число
- Координата Y: число
- Может возвращать текстовое представление вида “{X;Y}”

Необходимо создать три точки с разными координатами и вывести на экран их текстовое
представление.

#### Решение:
Создан класс Point в котором имеются:
- конструкторы принимаемые ничего,координаты точки, другую точку
- геттеры для координат точки
- сеттеры для координат и для изменения точки полностью
- преобразование в строку, которое выводит объект в виде “{X;Y}”

## Задание 2

#### Прямая.
Создайте сущность Линия, расположенную на двумерной плоскости, которая описывается:

- Координата начала: Точка
- Координата конца: Точка
- Может возвращать текстовое представление вида “Линия от {X1;Y1} до {X2;Y2}”

Для указания координат нужно использовать сущность Точка, разработанную в задании 1.1

#### Решение:
Создан класс Line в котором имеются:
- конструкторы принимаемые ничего, начальную и конечную точки, другую линию
- геттеры начала и конца линии
- сеттеры для установки начальной и конечной точки в виде как вставки точки так и координат
- преобразование в строку, которое выводит объект в виде “Линия от {X1;Y1} до {X2;Y2}”

## Задание 3

#### Студент.
Создайте сущность Студент, которая описывается:

- Имя: строка
- Оценки: массив целых чисел.
- Может возвращать текстовое представление вида “Имя: [оценка1, оценка2,…,оценкаN]”

#### Решение: 
Создан класс Student в котором имеются:
- конструкторы принимаемые имя, имя и оценки (так как студента без имени быть не может)
- геттеры имени, оценок (коппируемые ссылкой) и оценкок которые получаем без ссылок на оценки другого ученика
- сеттеры для установки имени, всех оценок, определенной оценки по индексу
- преобразование в строку, которое выводит объект в виде  “Имя: [оценка1, оценка2,…,оценкаN]”

## Задание 5

#### Студент отличник.
Измените сущность Студент из задачи 4.7. Новые требования включают:
- Студент может возвращать значение своей средней оценки (вещественное число). Если оценок нет, то среднее – 0.
- Студент может возвращать информацию о том, является ли он отличником или нет. Студент является отличником если у него есть хотя бы одна оценка и все оценки только отличные.

#### Решение: 
Копируем класс Student4 в класс Student5 и изменяем его:
- конструктор принимает только имя и оценки, если же оценки не запишут в конструктор при вызове, то будет пустой массив при выводе в строку
- геттеры имени, оценок (коппируемые ссылкой) и оценкок которые получаем без ссылок на оценки другого ученика
- сеттеры для установки имени, всех оценок, определенной оценки по индексу
- метод для средней оценки, который проходится по массиву оценок суммируя их, затем деля на количество оценок
- метод который смотрит отличник ли студент, используя метод выше он проверяет равна ли средняя оценка 5 или нет, если же оценок нет, то вернется тоже нет
- преобразование в строку, которое выводит объект в виде  “Имя: [оценка1, оценка2,…,оценкаN]”
